# Petshop

Este é um projeto simples de um site para um petshop, desenvolvido como parte das atividades do curso de ADS.

## Estrutura do Projeto

- `index.html`: Página inicial do site.
- `servicos.html`: Página de serviços oferecidos pelo petshop.
- `image/`: Pasta contendo imagens utilizadas no site.
- `video/`: Pasta contendo vídeos utilizados no site.

## Como Executar

1. Abra o arquivo `index.html` em um navegador web.
2. Navegue pelas páginas usando os links disponíveis.

## Tecnologias Utilizadas

- HTML
